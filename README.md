# Desafio: Módulo de Invoice (Sistemas Monolíticos)

Este projeto implementa o módulo de faturamento (Invoice) dentro de uma arquitetura de sistemas monolíticos, utilizando Domain-Driven Design (DDD) e o padrão Facade.

## Funcionalidades
- **Geração de Invoice:** Criação de faturas com múltiplos itens e endereço completo.
- **Consulta de Invoice:** Busca detalhada de faturas por ID.
- **Arquitetura:** Camadas de Domínio, Use Cases, Repositório (Sequelize) e Facade para integração.

## Como rodar
1. Instale as dependências:
```bash
npm install
```

2. Execute os testes:
```bash
npm test
```
